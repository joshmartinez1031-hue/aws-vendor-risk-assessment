# AWS Vendor Risk Assessment (GRC Project)

This project analyzes cybersecurity risks associated with using Amazon Web Services (AWS S3) as a third-party cloud storage vendor for Stripe.

## Project Objective
The goal of this project is to identify potential cloud security risks and misconfigurations that could expose sensitive financial and customer data.

## Framework Used
NIST Cybersecurity Framework (CSF)

## Key Risk Areas Identified
- Overly permissive IAM permissions
- Publicly accessible S3 buckets
- Improper encryption configuration
- Insufficient logging and monitoring
- Lack of multi-factor authentication

## Risk Mitigation Recommendations
- Enforce least privilege IAM policies
- Require MFA for privileged accounts
- Enable continuous monitoring and logging
- Implement encryption for sensitive S3 data
